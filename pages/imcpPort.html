Portfolio Website

Re!nvented

This portfolio website was created in Abode Dreamweaver and was the final project for the Interactive Media Certificate Program at San Diego Continuing Education.

The theme for the website was an art exhibition called Re!nvented. It was mainly inspired by the Art-o-Mat® - a retired cigarette vending machine that has been converted to vend art on wooden blocks or in boxes that are the size of a cigarette pack.

The website is non-responsive, a little outdated by now and contains flash animations but I think it still deserves a little attention in here. If you don't mind old, click here to explore the whole portfolio. Alternatively, just get a quick glance below.

The Concept

The first page of the portfolio is a facade of a museum that has a poster on the wall advertising the current exhibition. By clicking anywhere on the page, you enter the museum and land on the main page which is a hallway with five doors. They lead to the following rooms...

Print Media Room

By clicking on the image thumbnails in this upcycled cigarette vending machine you can explore graphic design and typography work from the print module.

Explore Print Media

Multimedia Room

The second room features an old television. By clicking on the buttons you can switch the channel (and the video).

Explore Multimedia

Web Design Room

This room is dedicated to interactivity and web design. It displays an old PC on which you can explore animations, web content, etc.

Explore Web Design

Library

The library is where you can find out more about the artist as well as contact information.

Cafeteria
